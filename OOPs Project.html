Object Oriented Programming in C++

Object-Oriented Programming - As the name suggests uses Object-Oriented Programming aims to implement real world entities like inheritance, hiding, abstraction, polymorphism, binding, etc. in programming.The main aim of OOP is to bind together the data and the functions that operate on them so that no other part of the code can access this data except that function.

There are some basics concepts that act as the building blocks of OOPs i.e.

  1. Class
  2. Objects
  3. Abstraction
  4. Encapsulation
  5. Inheritance
  6. Polymorphism
